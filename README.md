##  About the Project

gsap 스크롤 감지 애니메이션 기능을 활용하여, 사용자가 스크롤을 내릴 때마다
인터랙티브하게 반응하는 애플리케이션 홍보페이지를 디자인 기획, 퍼블리싱 구현하였습니다.

## Features

- 홈페이지 동영상 배경으로 보다 차별화되며 기업 이미지를 더욱 강력히 표현하였습니다.
