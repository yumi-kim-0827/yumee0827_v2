# [1] About the Project

기업 소개 홈페이지는 보통 원페이지 형식을 많이 활용합니다.   기업에 대한 비전, 서비스, 협력사에 대한 내용을 배치하였으며,   마지막 하단에 contact 입력, 방문자의 행동을 유도하였습니다.

## Features

- 홈페이지 동영상 배경으로 보다 차별화되며 기업 이미지를 더욱 강력히 표현하였습니다.
